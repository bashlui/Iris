# Iris

## What's Iris?

Iris is an educational iPad app that helps people understand vision conditions—myopia, hyperopia, and astigmatism—through simulation and interactive challenges. It turns empathy into a tangible experience: you can see the world through a live camera with real-time blur, complete tasks through simulated impairment, and explore how the eye works with a 3D model and theory.

Built for the **Swift Student Challenge 2026**.

---

## Features

- **Live vision simulator** — Split-screen camera view with Core Image blur. Compare clear vs simulated vision side by side. Adjust severity and (for astigmatism) axis. Supports myopia, hyperopia, and astigmatism.
- **Empathy challenges** — Three progressive challenges: find a specific emoji in a blurred grid, read a blurry prescription and pick the correct values, and type distant text through strong blur. Reduce impairment step-by-step if you can’t see.
- **Optical encyclopedia** — Theory tab with a SceneKit 3D eye model, condition details, causes, symptoms, and correction methods.
- **Vision tips** — Practical tips for eye care and healthy habits.
- **Accessibility** — Dynamic Type (ScaledMetric), VoiceOver labels and hints, high-contrast UI.
